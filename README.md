## 📌 About the Project

**Car Price Prediction** is an end-to-end Machine Learning web application that predicts the estimated selling price of a used car based on user-provided vehicle details.

The application leverages a trained Machine Learning regression model and provides an intuitive Streamlit interface, allowing users to enter vehicle specifications and receive an instant price prediction.

The project demonstrates the complete Machine Learning workflow, including data preprocessing, feature engineering, model training, evaluation, and deployment as an interactive web application. Streamlit is commonly used for rapidly deploying ML models as user-friendly web apps. :contentReference[oaicite:0]{index=0}

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/iammadhvi2207/Car-Price-Prediction-StreamLit.git

cd Car-Price-Prediction-StreamLit
```

---

### Create Virtual Environment

```bash
python -m venv venv
```

Activate

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

---

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

### Run the Application

```bash
streamlit run app.py
```
